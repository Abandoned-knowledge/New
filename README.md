# Основные команды git  
* touch "file_name.txt" - создаёт новый файл в текущей директории 
* mkdir "name" - создаёт новую папку в текущей директории
* rm "name" - удаление текущего файла
* rmdir "name" - удаление текущей папки, если она пуста
* rm -r "name" - рекурсивное удаление, удаляет сначала все элементы из папки, а потом саму папку  

# Git status  
данная команда проверяет статус файлов и имеет несколько значений:
* **untracked** файл добавление, его видит git, но никак не отслеживает.
* **tracked** файл был добавление с помощью команды **git add** и закоммичен с помощью **git commit**, эти в дальнейшем никак не отображаются
* **modified** уже отслеживаемый файл был изменен, то не добавлен и не закоммичен
* **staged** файл добален в репозиторий с помощью **git add**

## Примечание  
* файлы могут иметь сразу два состояние **staged** (tracked) + **modified**
